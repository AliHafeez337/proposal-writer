Job Description:

We're a vibrant and rapidly expanding team seeking a highly skilled and results-oriented Front End Software Engineer who can translate designs into exceptional user interfaces with speed and precision. If you have a strong command of modern front-end technologies and a proven track record of delivering high-quality UIs quickly, we want you on our team!

Responsibilities:

Develop and implement user interfaces and front-end components using HTML, CSS, and JavaScript.
Work extensively with modern front-end frameworks and libraries (e.g., React, Vue.js, Angular).
Translate UI/UX designs and wireframes into high-performing, responsive, and accessible web applications.
Optimize applications for maximum speed and scalability.
Collaborate closely with designers and backend developers to ensure seamless integration.
Write clean, well-structured, and maintainable code.
Troubleshoot and debug front-end issues effectively.
Participate in code reviews to maintain code quality and consistency.
Take ownership of front-end development tasks and deliver them efficiently within deadlines.

Requirements:

Deep expertise in modern JavaScript (ES6+).
Strong proficiency in HTML5 and CSS3.
Extensive experience with at least one major front-end framework/library (React, Vue.js, Angular). Experience with multiple is a significant plus.
Solid understanding of state management libraries (e.g., Redux, Vuex, Context API).
Experience with responsive design principles and techniques.
Familiarity with front-end build tools and workflows (e.g., Webpack, Parcel, npm, yarn).
Understanding of cross-browser compatibility issues and how to address them.
Knowledge of UI/UX principles and best practices.
Excellent problem-solving and debugging skills specific to front-end development.
Strong communication and collaboration skills.
Demonstrated ability to move quickly and deliver visually appealing and functional user interfaces at speed. Please provide examples in your application.
Experience with version control systems (e.g., Git).
Experience with testing frameworks (e.g., Jest, Enzyme, Cypress) is a plus.
Familiarity with accessibility standards (WCAG) is a plus.
